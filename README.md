# quizz-us

Platform to host a multiplayer quiz


Features
- Upload your own quiz questions
- Host a multiplayer quiz game
- Play the game with many players!

Requirements
- Go 1.20+
- Protobuf compiler (protoc)
- Node.js (for frontend development)
